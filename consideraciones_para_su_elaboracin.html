Consideraciones para su elaboración

Elaboración.

 

Recomendaciones para elaborar una lista de rangos:

 

1. Preparar una lista con los componentes o características importantes del comportamiento a observar. 

2. Seleccionar el tipo de escala con categorías para indicar el grado en el cual el comportamiento está presente. 

3. Incluir instrucciones adecuadas para asignar las categorías de la escala. 

4. Debe haber una escala independiente para cada característica, actitud o comportamiento que se desee medir. 

5. Tiene que ser como mínimo tres posibilidades en orden jerárquico (porque si son menos, es una lista de cotejo) por ejemplo:

Categoría cualitativa: Excelente, muy bueno, bueno, regular, suficiente, insuficiente y deficiente.

Categoría cuantitativa: Mucho, poco o nada.

Categoría de frecuencia: Siempre, casi siempre, a veces, casi nunca o nunca.

6. Para calcular su valoración se multiplica el valor máximo de la escala por el número de aspectos a observar.

7. Incluir un lugar para comentarios.
